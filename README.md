# 📊 Sales Data Analysis Project

## 🔹 Overview
Analyzed retail sales data to extract meaningful business insights using Python.

---

## 🧰 Tools Used
- Python (Pandas, Matplotlib)
- Jupyter Notebook

---

## 📂 Dataset
Sample Superstore dataset containing sales, profit, region, and category details.

---

## 📊 Analysis Performed
- Total Sales and Profit calculation
- Sales by Region and Category
- Top-performing States
- Profit analysis across categories

---

## 📈 Key Insights
- Technology category generated highest sales
- West region contributed maximum revenue
- Some categories show high sales but lower profit margins
- Profit varies across different categories

---

## 📁 Files
- analysis.ipynb → main analysis
- SampleSuperstore.csv → dataset
